Concepts

Skill Values

Raw Skill

Raw skill is a raw, linear measure of actions performed with a skill.   Every activity is worth some number of base skill points, modified by the difficulty of the task.

Base Skill

Base skill is calculated from Raw Skill, according to the following schedule:

For Base Skill to increase by 10, Raw Skill must double.

Raw Skill Base Skill
0 0
2048 10
4096 20
8192 30
16384 40
...
512K 90
1M 100
...

 

Effective Skill

Effective Skill is calculated from Base Skill.  Bonuses due to stats, magical effects, armor penalties, clothing, etc. are added to Base Skill to determine Effective Skill.
