Quick Start

  1. Download the lastest POL version. We'll call it POLXXX.zip here.
  2. Extract POLXXX.zip into a folder on your hard drive. For example, C:\POL.
  3. Open pol.cfg and edit the UoDataFileRoot= line to match the path where you have the UO client installed.
  4. Open login.cfg in your UO directory and change the LoginServer= to your IP or localhost. i.e. LoginServer=127.0.0.1,5003
  5. Open uo.cfg and comment out (by placing a semicolon [;] in front of the line) the lines: 
          UnicodeFont=On 
      UnicodeSpeech=On
    Or by setting these to Off.
  6. Open C:\POL\data\accounts.txt and change the admin account's password.
  7. Open up an MS-DOS prompt window ("dos box") and change to POL's directory and run 'POL'.
