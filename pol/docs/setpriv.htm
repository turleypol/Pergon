Privileges are things you can do that others may not be able to.

Settings are Privileges which are enabled.  Just because you CAN be invulnerable doesn't mean you have to be.

Settings/Privileges

Privilege Description
all Acts like all other privileges are granted
clotheany You can Clothe anyone (*)
dblclickany You can doubleclick any item from any range
dyeany You can Dye anything
hearghosts You can hear ghosts talk without spirit speak
invul You are Invulnerable (not yet implemented)
losany You have LOS to everything
moveany You can Move anything (*)
renameany You can Rename anyone
seeghosts You can see ghosts that aren't in warmode
seehidden You can see hidden players
seeinvisitems You can see invisible items
showadv Show advancement (raw skill values when they change)

 

MoveAny

You won't be able to move anything until the server re-sends the data to you.   Walk about 20 steps away, then back.

ClotheAny

You can only seem to add clothes; the client won't even try to remove clothing from someone else's paperdoll.
