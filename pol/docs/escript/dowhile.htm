DO..WHILE statement

General Form

DO
    statement
WHILE (expression);
  

NOTE, this does NOT yet use fully-bracketed syntax, so BEGIN/END must be used to group statements.

Examples

DO
    i := i + 1;
WHILE (i < 10);
   
DO
    BEGIN
        print "i=" + i;
        i := i + 1;
    END
WHILE (i < 10);
